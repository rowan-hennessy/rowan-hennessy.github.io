Project Overview

A ChatBot using PeakSpan Capital's MasterClass transcripts as context in RAG. This allows authorized users dynamic access to information discussed at events they potentially may have missed due to scheduling conflicts.

Implemented with LangChain and LangGraph framework for easy integration of tool-calling in agentic model.

Technologies Used

  • Python
  • LangChain
  • LangGraph
  • Ollama
  • HuggingFace
